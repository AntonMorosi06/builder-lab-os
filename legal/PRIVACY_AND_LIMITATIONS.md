# Privacy and Limitations

Builder Lab OS is currently a static prototype.

It does not store real user accounts, passwords, private uploads or cloud data.

Before becoming a real product, it will require:

- privacy review
- terms of service
- data model review
- security review
- storage policy
- deletion policy
- user consent model
- cookie/analytics review if analytics are added

Do not use this prototype to store private or sensitive user data.
