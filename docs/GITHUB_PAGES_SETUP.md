# GitHub Pages Setup

To publish Builder Lab OS as a public demo:

1. Open the repository on GitHub.
2. Go to Settings.
3. Open Pages.
4. Under Build and deployment, select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
5. Save.

Expected public URL:

```text
https://antonmorosi06.github.io/builder-lab-os/
```

If the page does not appear immediately, wait a few minutes and refresh.
