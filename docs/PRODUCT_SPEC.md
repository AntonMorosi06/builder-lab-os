# Product Specification — Builder Lab OS

## Product definition

Builder Lab OS is a project operating system for technical builders.

It is designed to help users organize technical work into a structured workspace composed of projects, repositories, study notes, evidence, milestones, videos and portfolio outputs.

## Target users

- computer science students
- engineering students
- makers
- robotics hobbyists
- cybersecurity beginners
- data science learners
- technical YouTubers
- GitHub portfolio builders

## Main modules

### Dashboard

Shows the global state of the user's technical ecosystem.

### Projects

Lists major technical projects with status, maturity and links.

### Evidence

Tracks screenshots, logs, reports, videos, commits and validation notes.

### Study

Connects learning activity to technical projects.

### Video Planner

Transforms project progress into YouTube episode ideas.

### Portfolio

Generates concise public-facing project summaries.

## Maturity labels

Builder Lab OS uses explicit maturity labels:

- planned
- prepared
- mocked
- validated-offline
- hardware-ready
- hardware-validated
- portfolio-ready
- production-ready

## Product philosophy

The product must help users avoid overclaiming.

A project should not be described as finished if it is only planned. A mockup should not be described as validated. Hardware should not be described as working without evidence.
