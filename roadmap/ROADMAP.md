# Builder Lab OS Roadmap

## v0.1 — Static Local Dashboard

Goal: create a local dashboard for organizing technical projects.

Features:

- project cards
- repository status
- evidence log
- roadmap overview
- YouTube episode planner
- portfolio summary section
- local JSON data

## v0.2 — Better Product Interface

Goal: make the product feel like a serious tool.

Features:

- better layout
- search/filter
- project detail pages
- status tags
- exported portfolio summaries

## v0.3 — GitHub-Oriented Workflow

Goal: connect the tool more closely to GitHub workflows.

Features:

- manual GitHub repository links
- commit/evidence checklist
- README quality checklist
- project maturity tracker

## v0.4 — Public Demo

Goal: publish a demo using GitHub Pages.

Features:

- clean landing view
- demo dataset
- screenshots
- documentation
- validation survey or contact method

## v1.0 — Product Validation Candidate

Goal: determine whether Builder Lab OS deserves real product development.

Requirements:

- feedback from real users
- clear use case
- polished demo
- repeatable workflow
- privacy and legal review
